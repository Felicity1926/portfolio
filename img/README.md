# Image Assets

This folder contains all image assets for the portfolio website.

## Required Images

### logo.png
- **Purpose**: Profile/logo image displayed on all pages
- **Usage**: 
  - Favicon (shortcut icon)
  - Hero section image on home page
  - About page profile image
- **Recommended specs**:
  - Format: PNG (with transparency)
  - Size: 200x200px to 400x400px
  - File size: Under 100KB

## How to Add Images

1. Replace or add `logo.png` in this folder
2. Images will be automatically linked from the HTML files
3. All image references in HTML already point to files in this directory

## Image References in HTML

The following files reference images:
- `index.html` - Uses `img/logo.png`
- `about.html` - Uses `img/logo.png`
- All pages use `img/logo.png` as favicon

## Best Practices

- Keep image file sizes optimized for web
- Use appropriate formats (PNG for logos/graphics, JPEG for photos)
- Ensure images are responsive and look good on all devices
